# StoneChat
由Stone编写 wocw作为精简与修改


一个简单的跨服聊天插件 使用文件作为服务器之间的缓存


使用方法：


给需要链接的服务端安装StoneChat 即 stonechat.jar文件放入plugins文件夹即可


将需要链接的服务端的文件夹放置在同一目录下 启动服务端 自动在上级目录生成Message.txt(缓存文件)
